# grill — Design Decisions

Non-derivable design decisions and constraints for this plugin. Read before making changes.

## 1. principles.md — relationship to project-quality

`skills/grill/references/principles.md` carries the simplicity values (1–6) that are
paraphrased and condensed from
`project-quality`'s `project-review-complexity/references/principles.md` (a 6-of-11
subset). The file is kept separate because plugins install independently — a reword in
`project-review-complexity` does not propagate here automatically.

**Keep the two files conceptually aligned when either changes.** If the simplicity
principles in `project-review-complexity/references/principles.md` are revised, apply
an equivalent update here.
