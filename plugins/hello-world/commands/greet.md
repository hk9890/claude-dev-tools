# Hello World Greeting

Welcome to the world of Claude Code plugins! This is a simple example slash command that demonstrates how to create custom commands.

## What This Command Does

This command will:
1. Greet you with a friendly message
2. Explain what plugins can do
3. Encourage you to explore further

## Output

Hello from the Hello World plugin!

This is an example slash command created to demonstrate how Claude Code plugins work.

Plugins can:
- Add custom slash commands like this one
- Create specialized agents for specific tasks
- Integrate external tools via MCP servers
- Set up hooks that respond to specific events

You can create your own plugins by following the structure of this example. Check out the README.md file in this plugin's directory for more details on how to get started!

Happy coding with Claude!
